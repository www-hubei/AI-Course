# 人工智能通识课 - 作业二 (HW02)

## 1. 任务说明
本项目包含两部分：关于 AI 驱动的未来学习中心建设的论文导读，以及一个调用 DeepSeek API 的 Chatbot 示例程序。

## 2. 任务一：论文导读
- **论文来源**：华夏, 薛睿, 等. 《基于AI大模型的高校混合式未来学习中心建设与实践》. 《实验室研究与探索》, 2026年第45卷第3期。
- **核心架构**：文中提出了 "1+1+N" 智慧架构，即一站式 AI 信息门户、AI 智能体学习中枢和 N 个教学实体空间。
- **成效数据**：实践表明该模式使跨专业创新课程开课率提升了 48.6%，大型设备利用率增长超过 10%。
- **配图方式**：导读中的架构图（fig1.png）、实验建设方案图（fig2.png）以及成效数据图（fig3.png）均为手动从原论文中截取并插入。

## 3. 任务二：Chatbot 示例代码
- **实现语言**：Python
- **调用模型**：DeepSeek-R1 (通过火山引擎 API 接入)
- **功能描述**：实现了用户输入文本问题，程序调用大模型并实时获取、打印回复的功能。
- **运行环境**：
  - Python 3.10 或以上版本。
  - 依赖安装：`pip install openai`。
- **运行方式**：
  - 在 `chatbot_deepseek.py` 中配置好您的 API Key 和 推理点 ID。
  - 在终端运行：`python chatbot_deepseek.py`。
